# The Boolean Types

- a Boolean type in Rust has two possible values: `true` and `false`. Booleans are one byte in size. The Boolean type in Rust is specified using `bool`.

```rust
fn main() {
    let t = true;

    let f: bool = false; // with explicit type annotation
}
```
